## Prerequisites

- Python 3.x

## Installation

1. Clone this repository to your environment :

```bash
git clone https://github.com/Vault-of-Jok3r/Casino-Directory.git
```

2. Navigate to the repository :

```bash
cd Casino-Directory
```

3. Run the script :

```bash
python main.py
```
